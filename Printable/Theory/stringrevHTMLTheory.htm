AIM: To reverse a string

THEORY:
C does not have a built in string type. Instead, string are represented by an array of characters where the element of the array after the last character of the string is a nul byte('\0')
When reversing a string, we can create a new string that is the reverse of the original string or we can mutate the original string to its own reverse. We shall be doing the later in this experiment.
The algorithm to reverse a string is written below:


ALGORITHM:

  1. Start
  2. Input str (A zero based indexed array of characters)
  3. len = length of string
  4. i = 0 , j = len - 1
  5. while i < j repeat steps 6 - 10
  6.    tmp = str[i]
  7.    str[i] = str[j]
  8.    str[j] = tmp
  9.    i = i + 1
  10.    j = j - 1
  11. Print str
  12. Stop
