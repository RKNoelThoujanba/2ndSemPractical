AIM: To display the first n terms of a Fibonacci sequence (f0 = 0, f1 = 1)


THEORY:
The Fibonacci sequence is a famous recursive sequence where Fn is defined as:
Fn = Fn-1 + Fn-2, F0 = 0, F1 = 1
So, we take two variables, first and second and set it to 0 and 1 respectively. Then we print first and second. We iterate from i = 0 to i = n - 1, where n is user supplied and for each iteration, we add first and second, then print them. After printing, we set first to second, second to sum and continue the iteration.

ALGORITHM:

  1. Start
  2. Input n
  3. if n <= 0 then goto step 12
  4. if n == 1 then print 0 and goto step 12
  5. first = 0, second = 1
  6. i = 2
  7. while i < n repeat steps 8 - 11
  8.     sum = first + second
  9.     Print sum
  10.     first = second
  11.     second = sum
  12. Stop
