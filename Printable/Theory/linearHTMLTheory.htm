AIM: To implement Linear Search


THEORY:
Linear Search is an algorithm used to search for a specific element in a sequence and in this case, an array. We take a user provided key and then iterate over each element of the array and compare it to our key. If the key matches the element, we have found our required index and can stop our search. If we have iterated through every element of the array and we still haven't found the key, the key does not exist in the array and we must stop our search.

ALGORITHM:

  1. Start
  2. Input n
  3. Initialize an array arr of size n (Zero based indexing)
  4. i = 0
  5. while i < n repeat steps 6 and 7
  6.     Input arr[i]
  7.     i = i + 1
  8. Input key
  9. i = 0
  10. while i < n repeat steps 11 and 12
  11.     if arr[i] == key then print key found at index i and goto step 14
  12.     i = i + 1
  13. Print key not found
  14. Stop
