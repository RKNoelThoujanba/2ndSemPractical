AIM: To determine is a user supplied integer is even or odd


THEORY:
Mathematically, an integer is said to be even if it is divisible by 2. If not, it is odd. So, if n % 2 is 0, n is even. if not, n is odd. (Here % operator gives the remainder when two integers participate in division)

ALGORITHM:

  1. Start
  2. Input n
  3. if n % 2 == 0 then
  4.     Print n is even
  5. else
  6.     Print n is odd
  7. Endif
  8. Stop
