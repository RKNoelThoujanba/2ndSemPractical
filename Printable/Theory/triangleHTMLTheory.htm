AIM: To print the following pattern:
4 3 2 1
3 2 1
2 1
1


ALGORITHM:

  1. Start
  2. Input n
  3. i = 1
  4. while i <= n repeat steps 5 - 10
  5.     j = n - i + 1
  6.     while j >= 1 repeat steps 7 and 8
  7.         print j
  8.         j = j - 1
  9.     print new line
  10.     i = i + 1
  11. Stop
