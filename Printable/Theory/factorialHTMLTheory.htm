AIM: To calculate the factorial of a user supplied positive integer

THEORY:
Mathematically a factorial may be defined as:
n! = n X (n - 1) X (n - 2) X ... X 3 X 2 X 1
or, n! = 1 X 2 X 3 X ... X (n - 1) X n
So we take a variable, say factorial and set it to 1. Then we iterate over i as i goes from 2 to n and multiply i to factorial in each iteration

ALGORITHM:

  1. Start
  2. Input n
  3. factorial = 1
  4. i = 2
  5. while i <= n repeat steps 6 and 7
  6.     factorial = factorial * i
  7.     i = i + 1
  8. Print factorial
  9. Stop
