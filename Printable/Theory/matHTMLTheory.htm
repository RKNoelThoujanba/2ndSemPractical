AIM: To multiply two user provided matrices A and B.


THEORY:
Let us consider two matrices Am x n and Bo x p
The necessary condition for the multiplication of A and B is: n = o.
The order of the resultant matrix is m x p, ie. (A X B) m x p = Am x n X o x p.
Below is the algorithm for matrix multiplication:

ALGORITHM:

  1. Start
  2. Input rowA, colA
  3. Initialize a matrix matA (zero based indexing) of dimensions rowA X colA
  4. i = 0
  5. while i < rowA repeat steps 6 - 10
  6.     j = 0
  7.     while j < colA repeat steps 8 and 9
  8.         Input matA[i][j]
  9.         j = j + 1
  10.     i = i + 1
  11. Input rowB, colB
  12. Initialize a matrix matB (zero based indexing) of dimensions rowB X colB
  13. i = 0
  14. while i < rowB repeat steps 15 - 19
  15.     j = 0
  16.     while j < colB repeat steps 17 and 18
  17.         Input matB[i][j]
  18.         j = j + 1
  19.     i = i + 1
  20. if rowB != colA then Print cannot multiply and goto 40
  21. Initialize a matix matProd (Zero based indexing) of dimmension rowA X colB
  22. i = 0
  23. while i < rowA repeat steps 24 - 31
  24.     j = 0
  25.     while j < colB repeat steps 26 - 30
  26.         k = 0
  27.         while k < colA repeat steps 27 and 28
  28.             matProd[i][j] = matProd[i][j] + matA[i][k] * matB[k][j]
  29.             k = k + 1
  30.         j = j + 1
  31.     i = i + 1
  32. i = 0
  33. while i < rowA repeat steps 34 - 39
  34.     j = 0
  35.     while j < colB repeat steps 36 and 37
  36.         print matProd[i][j]
  37.         j = j + 1
  38.     Print new line
  39.     i = i + 1
  40. Stop
