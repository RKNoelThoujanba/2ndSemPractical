Algorithm to display the firstg n terms of the fibonacci sequence



  1. Start
  2. Input n
  3. if n <= 0 then goto step 12
  4. if n == 1 then print 0 and goto step 12
  5. first = 0, second = 1
  6. i = 2
  7. while i < n repeat steps 8 - 11
  8.     sum = first + second
  9.     Print sum
  10.     first = second
  11.     second = sum
  12. Stop
