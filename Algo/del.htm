Algorithm to delete an element from an array



  1. Start
  2. Input n
  3. Initialize Array arr of size n (Zero based Indexing)
  4. i = 0
  5. while i < n repeat steps 6 and 7
  6.     Input arr[i]
  7.     i = i + 1
  8. Input deletionIndex
  9. i = deletionIndex
  10. while i < n - 1 repeat steps 11 and 12
  11.     arr[i] = arr[i+1]
  12.     i = i + 1
  13. n = n - 1
  14. i = 0
  15. Print new array:
  16. while i < n repeat steps 17 and 18
  17.     print arr[i]
  18.     i = i + 1
