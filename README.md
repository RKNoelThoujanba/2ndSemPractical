<h1 > Build Instructions: </h1>
<br>

<p>

cd build <br>
cmake ../ <br>
cmake --build . --config release [-j no of threads to be allocated to compile] <br>

<br><br>

executables should be in 2ndSemPractical/build/Release <br>
</p>
